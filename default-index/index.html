错误笔记——编辑cpp链接不到动态库

发表于 2022-04-14 分类于 错误笔记

错误提示：链接不到动态库，libglpk.so.40不存在

解决：

  1. 用“ldd 可执行文件”查看是否缺少libglpk.so.40，发现libglpk.so.40==>>no file
  2. 在网上没有找到下载libglpk.so.40的资源，因为之前安装了glpk，不应该没有libglpk.so.40呀！
  3. 在lib中也没有搜到libglpk.so.40
  4. 用which搜索libglpk.so.40，找不到啦
  5. 用find搜索libglpk.so.40，找到libglpk.so.40的路径啦
  6. vim /etc/ls/so/conf
  7. 在文件末尾加上libglpk.so.40的路径
  8. ldconfig
  9. 重新make *.cpp
  10. 成功！
